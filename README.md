# Amatda web-memo-site
+ vue.js /nuxt 프레임워크 이용하여 일정 및 스케줄 관리 할 수 있는 싱글페이지 프로젝트를 구현했습니다.
+ 현재 데스크탑 해상도 기준으로 개발되어있습니다.(모바일 반응형 미적용)
